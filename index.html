Aqui estão alguns dos meus projetos

ECOLETA

O Ecoleta foi um projeto desenvolvido durante a Next Level Week, uma semana de curso intensivo promovida pela Rocketseat. O projeto consiste em um sistema web que cadastra, e busca pontos de reciclagem pelo Brasil.

ELECTRON AUTO UPDATE APP

Esta aplicação consiste em atualizar automaticamente aplicações buildadas com o Electron. A aplicação busca de maneira automatica novas releases no repositório git da aplicação. Após esse processo, o usuário é informado de que uma nova versão está disponível, bastando apenas reiniciar a aplicação

RELÓGIO POMODORO

A página @codar.me do instagram propôs um desafio para treinar os conhecimentos em Javascript. O desafio era criar o cronometro Pomodoro(uma técnica de concentração muito usada), que consiste em um contador de tempo que se intercarla entre 25 e 5 minutos. Sendo os 25 um tempo de foco, e os 5 um tempo de descanso.

PREVISÃO DO TEMPO

Uma página web que mostra a previsão do tempo para uma localidade de escolha do usuário. A aplicação consulta uma API pública para realizar a tarefa.
